# Project Roadmap & TODOs

This document tracks planned features, improvements, and technical debt for the ACM Recruitment Portal.

## Future Features

### Student Experience
- [ ] **UI/UX Overhaul:**
  - Make interface cleaner and self-explanatory.
  - Simplify navigation and forms.
- [ ] **Interview Scheduling:**
  - Student slot booking system.
  - Jitsi Meet integration for in-platform interviews.
  - Interview staging/waiting area.
- [ ] **Auth UI Improvement:**
  - Show "Sign in and sign up disabled by admin" message when both auth methods are turned off.

### Department Administration
- [ ] **Notes System:**
  - Enhanced candidate notes content.
  - Better visibility/access to notes.
- [ ] **Interview Management:**
  - Set up available time slots.
  - Public/Private slot visibility toggles.
- [ ] **Data Export:**
  - Export applicant data to CSV/Excel.
  - Export round results.
- [ ] **Bulk Actions:**
  - Bulk accept/reject candidates.
  - Bulk move candidates to next round.
- [ ] **Advanced Filtering:**
  - Filter by custom question responses.
  - Search by skills or keywords in cover letters.

### Security & Admin
- [ ] **Activity Logs:**
  - Track admin actions (who changed what).
- [ ] **Password Reset:**
  - "Forgot Password" flow for email accounts.
- [ ] **Role Granularity:**
  - More granular permissions for department admins.
